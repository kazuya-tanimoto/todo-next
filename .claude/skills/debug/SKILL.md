# デバッグ手順

エラーやテスト失敗が発生した場合:

1. **まず根本原因を調査する**
   - エラーメッセージを正確に読む
   - 環境情報を確認する（Node.jsバージョン、OS、依存パッケージ等）
   - 関連するドキュメントやchangelogを調べる

2. **原因を特定してから修正する**
   - 推測による修正を連続して試みない
   - 2回連続で同じ種類のエラーが出たら、立ち止まってアプローチを見直す

3. **修正は最小限にする**
   - 根本原因に対応する変更のみ行う
   - 関連しないライブラリの変更やAPI変更で回避しない
